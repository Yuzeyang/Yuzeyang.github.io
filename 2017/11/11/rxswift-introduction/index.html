RxSwift 入门

2017年11月11日

首先 RxSwift 是什么？

RxSwift 是 ReactiveX（Rx）的 Swift 版本。而 Rx 是一个响应式编程框架，响应式编程是一种面向数据流和变化传播的编程范式，这与以往我们用命令式编程不同的是它能够将数据的变动经过数据流传播给受影响的对象，举一个简单的例子：a = b + c ，在命令式编程中，一次赋值之后，后续 b 与 c 的变动不再影响 a ，除非再执行一次该赋值语句，而响应式编程则会影响。

这也就是 Rx 的优势所在：its intention is to enable easy composition of asynchronous operations and event/data streams. 能够简化异步操作和事件/数据流。

那它是如何简化的呢？举两个例子：

  • 图片下载

以往我们会异步处理图片数据获取，然后切换到主线程设置图片

在 RxSwift 中可以通过 observeOn 将切换线程的操作提取出来，线程中的操作写在 observeOn 之后

  • 关键字搜索（官方例子）

以往通过 Delegate 获取搜索关键字的改变，进而做数据的请求展示。

而使用rx之后，可以通过subscribe订阅想要的属性，当属性的值改变之后执行action
